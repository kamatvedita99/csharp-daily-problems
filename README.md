# C# Daily Problems

Daily C# coding problems focused on financial domain scenarios — trade enrichment, portfolio calculations, market data processing, async patterns.

Each file contains:
- Problem statement in comments
- Clean solution
- Key learnings at the bottom

| 04 | Error Handling | 8/10 | ArgumentException, structured logging, custom exceptions |
